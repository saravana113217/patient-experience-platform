# 🏥 Patient Experience Analytics Platform

An AI-driven real-time patient feedback analysis system designed to improve hospital service quality through intelligent complaint monitoring and automated alerts.

## 🚀 Features

- Real-time patient complaint submission
- AI-based sentiment detection
- Automatic severity classification
- Critical issue alerts
- Notification system
- Complaint tracking dashboard
- Department-wise analytics

## 🧠 AI Component

The platform uses a trained machine learning model to analyze feedback sentiment and classify complaints into severity levels for faster response.

## 🛠️ Tech Stack

- Python (Flask)
- HTML, CSS, JavaScript
- SQLite Database
- Machine Learning Model (scikit-learn)
- REST API

## ⚙️ How It Works

1. Patient submits complaint
2. System analyzes sentiment using AI
3. Severity level is assigned automatically
4. Critical cases trigger alerts
5. Data stored for analytics and monitoring

## 🎯 Use Case

Helps hospitals detect dissatisfied patients early, reduce risks, and improve overall service quality.

## ▶️ Run Locally

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open in browser:

http://127.0.0.1:5000
